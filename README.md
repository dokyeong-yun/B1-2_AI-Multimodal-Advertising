# 📄 B1-2. AI 멀티모달 기반 브랜드 광고 제작 파이프라인 및 가이드라인 패키지

---

## 1. 개요 및 가이드라인 수립 목적 [평가기준 1]
* **과업 정의:** 기획 의도(스토리보드)가 생성형 AI 프롬프트를 거쳐 최종 10초 이내의 고품질 MP4 광고 영상으로 구현되기까지의 전체 멀티모달(시각·청각) 제작 파이프라인 설계 및 검증.
* **타겟 사용자:** 1인 교육 콘텐츠 기업가, 중고등 수학 마케팅 담당자, 또는 최소한의 자원으로 시네마틱 브랜드 경험을 구축하고자 하는 소규모 크리에이티브 디렉터.
* **핵심 과제:** 멀티모달 생성 시 빈번히 발생하는 3대 한계인 **① 캐릭터/화풍 불일치, ② 크레딧/비용 조기 소진 리스크, ③ 오디오-비주얼 미스매치**를 기술적 프롬프트 엔지니어링 및 대체 파이프라인 구축을 통해 해결함.

## 8. 심층 인터뷰 및 멀티모달 장기 운영 전략 [평가기준 14, 15]

### Q1. 멀티모달 제작 환경에서 각 생성 도구(텍스트, 이미지, 비디오, 오디오) 간의 결과물을 결합할 때, 가독성과 시각적/청각적 톤앤매너 일관성을 프롬프트 차원에서 강제하는 구체적인 엔지니어링 방법은 무엇인가요? [평가기준 14]
* **답변:** 도구 간의 파편화된 문맥을 일치시키기 위해 **'공통 키워드 토큰 앵커링(Token Anchoring)'**과 **'메타데이터 크로스 주입'** 전략을 적용합니다. 
  1. 이미지 생성 단계에서 성공한 핵심 스타일 단어 3가지(예: `cyberpunk, neon rain reflections, cinematic low-key lighting`)를 고정 마스터 토큰으로 추출합니다.
  2. 비디오 변환 도구(Runway 등)에 텍스트 프롬프트를 입력할 때 이 마스터 토큰을 반드시 서두에 재주입하여 이미지의 광원과 질감이 영상 모션 속에서도 깨지지 않도록 제어합니다.
  3. 오디오 도구(Suno)의 스타일 태그 영역에도 단순히 'Cyberpunk BGM'이라고 적는 대신, 비주얼 메타데이터에서 기인한 청각적 요소인 `[Lo-fi beats, ambient wet rain sound effect, dark cinematic sub-bass]`를 명시적으로 크로스 주입함으로써 시각적 네온 이미지와 청각적 사운드트랙의 톤앤매너를 프롬프트 구조상에서 100% 동기화시킵니다.

### Q2. 대화 세션이 10턴 이상 길어지거나 네이토 플랫폼의 창이 강제로 닫혀 단기 기억(Context Window) 망각 현상이 일어날 때, 기 수립된 브랜드 가이드라인과 씬별 필수 필드 제약 조건을 AI가 잃어버리지 않게 유지하는 프롬프트 제어 방안은 무엇인가요? [평가기준 15]
* **답변:** 장기 멀티모달 협업 세션에서 발생하는 AI의 지침 망각 및 탈락 문제를 해결하기 위해 다음의 세 가지 시스템 프롬프트 엔지니어링 기법을 적용합니다.
  1. **압축형 세션 가이드북 토큰화(Session Condensation Token):** 매 5턴마다 대화 흐름 속에서 확정된 에셋 스펙(예: 16:9 비율, H.264, `--sref 12345`)을 하나의 `[CURRENT_PIPELINE_STATE]` 블록으로 압축 요약하여 AI가 스스로 단기 메모리를 리프레시하도록 백엔드 제어를 수행합니다.
  2. **시스템 앵커 리마인더(System Anchor Reminder) 자동 배치:** 유저가 새로운 씬 생성을 요청하거나 프롬프트 수정을 요구할 때마다, 유저 입력 데이터 바로 뒤에 `[체크리스트 리마인더: 당신은 NeoSeoul 브랜드 디렉터이며, 모든 출력은 씬 번호/길이/목표 메시지/입력 프롬프트 원문 필드를 반드시 포함해야 합니다]`라는 지시형 닻(Anchor) 프롬프트를 자동 결합하여 모델의 어텐션(Attention) 분산을 막습니다.
  3. **메모리 아키텍처 레이어 분리(Memory Layer Separation):** 기획 문서의 고정 속성(브랜드 아이덴티티)과 유동 속성(씬별 프롬프트 실험 결과)을 완벽히 분리하여, 대화 내용이 길어지더라도 상위 시스템 명령 레이어가 하위 대화 데이터 로그에 의해 덮어씌워지지 않도록 프롬프트 서식을 격리 운영합니다.
