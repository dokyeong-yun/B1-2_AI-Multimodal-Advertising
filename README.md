## 7. 핵심 기술 원리 적용 (추가 필수)

### 7-1. [평가기준 12] T2I와 I2V(T2V) 기술적 차이 및 선택 기준
* **T2I (Text-to-Image):** 텍스트 프롬프트의 의미적 벡터를 시각적 픽셀로 전환하는 기술로, 고정된 구도와 정밀한 브랜드 톤앤매너(그리드, 패키지 디자인)를 '정지 상태'에서 완벽하게 제어하기 위해 첫 소스 확보 단계로 선택했습니다.
* **I2V (Image-to-Video):** T2I로 완성된 고품질 이미지의 공간적 맥락을 유지한 채 시간 축에 따른 프레임 간 픽셀 변화를 예측하는 기술입니다. 처음부터 텍스트만으로 영상을 만드는 T2V 방식은 형태가 무너지거나 화풍이 튀는 현상이 심하므로, 브랜드 일관성을 철저히 지키기 위해 **[T2I 선 제작 → I2V 후 변환]** 구조를 바인딩 규칙으로 선택했습니다.

### 7-2. [평가기준 13] 도구 선택 및 생성 우선순위 기전
본 프로젝트의 생성 파이프라인 구축 시 우선순위는 **[ 품질(Quality) > 제어력(Control) > 비용(Cost) > 속도(Speed) ]** 순으로 책정되었습니다.
* **이유:** 35moment는 프리미엄 감성 코스메틱 브랜드이므로, 생성 속도가 다소 느리거나 크레딧 비용이 더 들더라도 미학적 '품질'과 브랜드 규격을 유지할 수 있는 '제어력(Style Reference 기능 등)'을 가진 최신 엔진(네이토 내 Midjourney 및 Runway Gen-3 등)을 최우선으로 선택하여 기획 의도의 왜곡을 방지했습니다.

## 8. 심층 인터뷰 답변 항목 (추가 필수)

### 8-1. 서로 다른 도구 결과물의 통합 보정 기준 (인터뷰 1)
네이토 내의 여러 AI 엔진(Runway, Kling 등)에서 생성된 소스들은 해상도와 미세한 색감 차이가 발생합니다. 이를 통합하기 위해 영상 편집기(CapCut)에서 해상도를 **1080p(1920x1080)**로 고정 인코딩하고, 무채색(씬 1)과 오렌지색(씬 2, 3)의 대비가 극대화되도록 LUT 색보정 값을 일괄 적용하여 시각적 이질감을 상쇄합니다.

### 8-2. 다른 도구 사용 시 결과물 예측 및 대비 (인터뷰 2)
만약 네이토 내부 엔진 대신 외부 Flux.1(이미지)이나 Luma(영상)로 도구를 전면 교체할 경우, Flux의 높은 텍스트 렌더링 능력 덕분에 제품 패키지의 로고 표현은 더 선명해질 수 있으나, Luma의 모션 특성상 카메라 무빙이 과격해져 미니멀한 브랜드 톤이 깨질 수 있습니다. 이에 대비해 프롬프트에 `cinematic static shot, extreme slow motion` 제어어 가중치를 높이는 대안을 설계했습니다.

### 8-3. 크레딧 부족 시 파이프라인 및 전략 변경안 (인터뷰 3)
생성 크레딧이 50% 이상 급격히 부족해지는 상황이 발생할 경우, 비디오 변환(I2V) 횟수를 전면 제한하는 전략을 취합니다. 대신 이미 확보된 고품질 이미지(`scene01_keyvisual.png` 등)에 편집기의 기능인 디지털 줌인/줌아웃(Keyframe 2D 모션) 및 디졸브 트랜지션을 적용하여 영상 크레딧 소모를 0으로 방어하면서도 시각적 움직임을 구현해 낼 것입니다.

### 8-4. 타겟 플랫폼 변경으로 영상 길이 축소 시 재구성 안 (인터뷰 4)
만약 10초 광고를 5초 숏폼 범퍼광고로 축소해야 한다면, 전개부가 길어지는 '씬 1(지친 일상)'의 비중을 1초로 극단적으로 줄이거나 생략합니다. 그리고 곧바로 '씬 2(오렌지빛 전이)'의 강렬한 비주얼로 시작(2초)하여, 35moment의 브랜드 로고와 슬로건이 노출되는 '씬 3(제품 및 CTA)'을 3초 동안 길게 유지함으로써 메시지 타격감과 브랜드 각인 효과를 동시 확보할 것입니다.
